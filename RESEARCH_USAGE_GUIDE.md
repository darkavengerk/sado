# 한국 전통 귀신·요괴 조사 — 사용 가이드

## 목적

`korean_spirits.md`는 조선백귀야행(朝鮮百鬼夜行) 소설 프로젝트의 **민속학적 기초 자료**입니다.  
이 가이드는 시놉시스·캐릭터·플롯 설계 팀이 이 자료를 효과적으로 활용하기 위한 방법을 제시합니다.

---

## 문서 구조

### 1. 개요 섹션
- **시대 배경:** 조선 후기 영조 치세(1724-1776)
- **문화적 맥락:** 임진왜란·병자호란 이후의 사회적 변화와 민간신앙의 공고화
- **야행(夜行) 개념:** 음력 7월 15일(백중) 자정의 귀신 행렬

**활용법:**  
시놉시스의 시대적 배경과 분위기 설정에 직접 인용 가능. 사도세자 사건(1762)과의 연결성 주목.

### 2. 존재 목록 (10종)

각 존재마다 동일한 구조로 정보 제공:

| 섹션 | 용도 |
|---|---|
| **외모** | 캐릭터 비주얼 설계, 의상·메이크업 참고 |
| **능력** | 플롯 디바이스(곤경 해결 수단), 갈등 구조 설정 |
| **약점** | 캐릭터 한계 설정, 적대자 전술 구성 |
| **출몰 장소** | 장면 설정, 서사 공간 배치 |
| **조선 후기 기록 사례** | 역사적 신뢰성 확보, 대사·상황의 모티프 |
| **스토리텔링 포인트** | 캐릭터의 서사적 역할, 주인공과의 관계 |

## 활용 시나리오별 참고법

### 시나리오 1: 캐릭터 설계 (be-dev-1 진행 중)

**문서의 어느 부분을 봐야 하나?**
- "스토리텔링 포인트" 섹션 읽기 (각 존재 마다)
- "존재별 서사 역할 제안" 테이블 (224-236행) 참조
- 각 캐릭터의 "외모" 항목으로 비주얼 설정

**예시:**
```
도깨비 캐릭터를 설계할 때:
- 외모: "방망이를 든 커다란 체구" → 신체 크기·소품 설정
- 능력: "방망이로 무엇이든 만들어내는 능력" → 이 캐릭터는 주인공의 물질적 문제 해결 도구
- 약점: "새벽 닭 울음소리" → 야행이 여명으로 끝나는 시간 제약 설정
- 역할: "코믹 릴리프, 지하 경제의 중개인" → 톤 설정, 관계성 설정
```

### 시나리오 2: 플롯 설계

**문서의 어느 부분을 봐야 하나?**
- "능력"과 "약점" 항목으로 갈등 구도 설계
- "출몰 장소"로 장면 배치 구성
- "핵심 주제 제안" (238-242행) 으로 플롯 뼈대 설계

**예시:**
```
"물귀신 대신자 원칙" (물귀신이 익사 희생자를 요구함) →
주인공이 한강을 건널 때 도덕적 딜레마 상황 설정 가능:
- 자신 대신 누군가를 희생시킬 것인가?
- 다른 방법으로 통과할 것인가?
→ 이것이 주인공의 성격과 선택을 드러내는 중요 장면
```

### 시나리오 3: 시놉시스 작성

**문서의 어느 부분을 봐야 하나?**
- "세계관 설정" (218-221행)
- "핵심 주제 제안" (238-242행)
- "존재별 서사 역할 제안" (223-236행) 테이블

**예시:**
```
시놉시스의 "갈등 구도" 섹션:
→ 구미호(정보 브로커)와 산귀(외부 세력)의 대립
→ 이무기(최종 보스)와 야행의 질서 체계 간의 충돌

시놉시스의 "주제" 섹션:
→ "한(恨)의 정치학" (사도세자 사건의 잔영)
→ "용서와 해원" (원한 해결을 통한 초월)
```

## 역사적 신뢰성 확보 방법

### 1차 문헌 활용

이 문서에 인용된 고전문헌들은 **실제 조선 시대 기록**입니다:

| 문헌 | 시대 | 활용 방법 |
|---|---|---|
| 『어우야담』 | 17세기 초 | 귀신의 심리·욕망 묘사 |
| 『청구야담』 | 조선 후기 | 민간 관습·의례 설정 |
| 『조선왕조실록』 | 영조 대 | 시대 배경의 역사적 정확성 |
| 『한중록』 | 혜경궁 홍씨 | 사도세자 사건의 심리적 맥락 |

**예시:**
```
처녀귀신 장면을 쓸 때:
"영조 31년(1755년) 명혼(冥婚) 의식 금지령" 사건을 배경으로
→ 주인공이 이 금지령 때문에 처녀귀신을 도와주지 못하는 갈등 상황 설정 가능
→ 역사와 플롯이 유기적으로 연결됨
```

### 고증 체크리스트

캐릭터나 장면을 설계할 때:

- [ ] 해당 귀신의 "조선 후기 기록 사례" 섹션을 읽었는가?
- [ ] 그 시대의 실제 믿음과 의례를 반영했는가?
- [ ] 과도한 현대적 해석을 섞지 않았는가?
- [ ] 참고 문헌 목록의 원문을 확인할 수 있는가?

## FAQ

**Q: 구미호가 실제로 900년 이상 살아있을 수 있나요?**  
A: 이 문서는 조선 후기 **민간신앙**을 정리한 것이지, 역사적 사실이 아닙니다. 설화에서는 이렇게 믿어졌다는 점이 중요합니다.

**Q: 야행(夜行)은 역사 기록에 있나요?**  
A: 일본의 "백귀야행"(百鬼夜行) 전설에서 영감을 얻은 개념으로, 조선 후기의 실제 믿음을 기반으로 소설적으로 구성했습니다.

**Q: 각 귀신의 약점(개 피, 팥 등)은 왜 그렇나요?**  
A: 오래된 민간신앙에서 비롯된 것으로, 단군 신화(쑥과 마늘)나 음양오행 사상과 연결됩니다. 문헌에 따라 근거가 다릅니다.

**Q: 이 설정을 완전히 무시하고 새로 만들 수 있나요?**  
A: 프로젝트의 핵심은 "조선 후기의 역사와 민속을 바탕으로 한 창작"이므로, 최대한 이 기초를 존중하되 필요한 수정은 협의를 통해 진행합니다.
