## 1. 차원문의 성질

### 1-1. 정의 및 특성

**차원문(次元門)**: 현대와 조선 시대를 잇는 초자연적 통로. 한(恨) 에너지로 열리는 시공간 균열.

| 항목 | 설정 |
|------|------|
| **형태** | "빛의 반대" — 빛이 빠져나가는 어둠의 구멍. 공간이 자신을 들여다보는 듯한 시각적 현상 |
| **열림 조건** | 한(恨) 에너지(특히 뒤주 목재 파편 같은 원사 유물)와 민감한 인간의 접촉 |
| **폐쇄 메커니즘** | 자동 폐쇄. 통과자가 목적지에 도착하면 문은 뒤따라 닫힘 |
| **두 방향 통과** | 현단계에서는 편향적—현대→조선은 가능하나, 조선→현대 통과는 미정(설정 유보) |
| **위치 고정성** | 창덕궁 비원(秘苑) 인근이 중심 게이트 위치. 다른 장소의 추가 게이트 가능성 있음 |

### 1-2. 에너지원: 한(恨)의 집합

차원문은 **한(恨) 에너지의 물리화(物理化)**로 이해됨. 한은 다음의 집합:

- 억울하게 죽은 자들의 원혼(怨靈)
- 역사에서 지워진 자들의 기억
- 권력에 의해 소멸된 목소리들
- 조선 사회 구조 속 억압된 자들의 정한(貞恨)

**뒤주 목재 파편**: 1762년 사도세자가 갇혀 죽은 뒤주의 물질적 잔여. 뒤주 속 원혼의 한(恨)이 수백 년간 응축된 물건. 파편 자체가 작은 게이트 열쇠처럼 기능.

## 2. 3단계 통과 경험

### 2-1. 1단계: 일상의 균열 (발단)

**시간**: 오후 5시, 장마 직후 서울 여름  
**장소**: 창덕궁 담벼락 꺾이는 지점

**감각 현상:**
- **촉각**: 손바닥의 뜨거움 — 지퍼백을 통해서도 느껴지는 나무 조각의 발열
- **청각**: 서울 소음의 뭉개짐 — 마치 수중에서 들리는 것처럼 변질
- **후각**: 차갑고 정체된 공기 — 습한 여름 열기의 급격한 소퇴
- **시각**: 담벼락 너머 비원의 나무들 사이에서 "빛의 부재" — 빛의 반대로 열리는 어둠의 구멍

**주인공의 반응**: 움직일 수 없음. 발이 나아가거나 빨려들어가는 혼동된 감각.

---

### 2-2. 2단계: 시공간 왜곡 (통과)

**시간**: 측정 불가(1초 미만에서 영원에 이르는 주관적 시간 왜곡)

**감각 순서 및 강도**:

1. **후각 선행** (가장 먼저 침습)
   - 오존(雷電 직전의 공기) 냄새: 코끝을 쐐기처럼 파고듦
   - 소나무 수지(새 수액): 옻칠한 고궁이 아닌 생생한 신선함
   - 철(鐵)과 피의 냄새: 혀 뿌리까지 밀려옴 → 위장 뒤틀림

2. **촉각 침략** (신경계 전체)
   - 중력 방향 소실: 아래와 위가 동시 존재→동시 소멸
   - 추락과 상승 구분 불가
   - 좁은 것을 통과하는 압력: 새의 뼈가 눌리는 감각
   - 양 손바닥 냉각→발열 교체: 냉과 열이 번갈아 침략 (신경이 구분 포기)

3. **시각 분해** (파장 분리)
   - 색의 분리: 파란색이 파란색에서 벗겨짐
   - 색의 반전: 파란색→노란색, 흰색→검정
   - 하늘 찢어짐: 찢어진 하늘 너머 "텅 빈 것"(어둠도 없는 상태)

4. **청각 왜곡** (음파 파괴)
   - 서울 소음의 늘어남 → 테이프 고속 되감음 같은 왜곡
   - 침묵의 찰나(완전 무음)
   - 북소리: 신호 같기도 애도 같기도 한 규칙 없는 타격음
   - 여자의 울음소리: 수십 명에서 수백 명이 동시 흐느낌

**주인공의 감정 상태**: 극도의 공포, 판단력 상실, 죽음과 정신 붕괴의 혼동

**물리적 신호**: 나무 조각이 맨살에서 뜨겁게 진동. 진동이 "앞으로"라는 메시지 같이 들림.

### 2-3. 3단계: 1882년 한양 도착 (착지)

**시간**: 1882년 여름 오후, 임오군란 직후  
**장소**: 창덕궁 후원(後苑) 근처 황토 지표

**감각 착지 순서**:

1. **촉각 착지**: 무릎부터 땅에 닿음 → 점토 냄새나는 황토의 무거움 → 손 짚음 → 태양열 담은 황토의 뜨거움이 손바닥 살을 지남

2. **후각 층상 쌓임** (가장 강렬):
   - 말의 배설물 (가축 냄새)
   - 불에 탄 목재 (건물 소각)
   - 화약 연기 (매캐한 황 냄새)
   - 이 모든 냄새가 층층이 쌓여 숨통을 눌림

3. **시각 재형성**:
   - 하늘이 낮음 (고층 건물 없음, 신호등 없음)
   - 연기가 오후 햇살을 탁하게 만듦
   - 지평선: 기와지붕과 흙담만 이어짐
   - 창덕궁 후원의 나무들은 있으나, 그 너머는 21세기가 아님

4. **청각 현실화**:
   - 사람들의 발소리 (짚신, 샌들)
   - 옷자락 끌리는 소리
   - 질주하는 발걸음 소리
   - 비명과 울음소리
   - 말발굽 소리 (진동이 땅 밑에서 올라옴)

**역사적 고증 요소**:
- 짚신(草履) 신은 발
- 삼베 바지 끝이 땅에 끌림
- 흰 무명 치마를 붙잡은 여자들
- 등짐을 진 남자들
- 갑주를 입은 **청나라 병사들** ← 임오군란(1882) 직후 청 군대 주둔의 시각적 증거

**초자연 현상**:
- 불이 피어오르는 아지랑이처럼 투명한 형상
- 희고 투명한 무엇 (윤곽 흔들림)
- **눈만 고요하고 단단함** — 수백 년을 기다려온 눈으로 주인공을 바라봄
- **신원**: 사도세자(思悼世子)의 원혼

## 4. 주인공-차원문-원혼의 관계구조

```
┌─────────────────────────────────────┐
│  뒤주 목재 파편 (물리 게이트)      │
│  ↓                                   │
│  한(恨) 에너지 응축체                │
│  ↓                                   │
│  이수연(주인공) 민감성 + 접촉        │
│  ↓                                   │
│  차원문 개방                         │
│  ↓                                   │
│  1882년 한양 도착                    │
│  ↓                                   │
│  사도세자 원혼과의 대면              │
│  ("이제 돌아갈 수 없다")             │
└─────────────────────────────────────┘
```

**주인공의 깨달음**:
- "여기가 1882년이다" — 1882년이 실제임을 온몸으로 인식
- "이제 돌아갈 수 없다" — 물리적 귀환 불가능성 자각
- "이것이 처음부터 자신이 있어야 할 자리였다" — 운명적 소환 자각
