## 상세 타임라인

### 【1762년】 계약 체결 — 임오화변(壬午禍變)

**역사 사건:**
- 음력 윤5월 21일, 영조 38년
- 영조가 세자 사도를 뒤주(木庫)에 가두어 8일 만에 죽임
- 노론 강경파 정치 음모와 부자 갈등의 결과

**세계관 사건:**
- 사도의 원혼, 일반 원한 크기를 초월한 "삼중 한(恨)" 발생
- 저승사자가 망설임: 이런 크기의 원한은 처리 불가능
- 흑마술사 나가사가 뒤주 속 7박 8일 동안 접근
- **계약 체결:**
  - 사도: 120년간 이승 방랑 → 원혼 기억 수집 → 한(恨) 에너지 축적
  - 나가사: 120년간 영계 결계 유지 + 저승사자 저지
  - 대가: 120년 후 축적된 에너지의 절반을 나가사에게 양도

**에너지 상태:** 0% → 계약 체결 완료, 이승 잔류 가능

**인물:**
- 사도세자(원혼): 왕자, 불의의 죽음
- 나가사: 흑마술사, 이승-저승 경계 탐구자
- 저승사자: 원혼 관할 신령, 망설임

### 【1776년】 정조 즉위

**역사 사건:**
- 정조(正祖) 즉위 (사도세자의 아들)
- 정조는 즉시 아버지의 명예 회복 시도 시작

**세계관 사건:**
- 사도의 원혼, "아들이 나를 기억한다"고 느낌
- 첫 조기 해제 희망 발생

---

### 【1783년】 사도세자 추존 (장헌세자로 추존) → **에피소드 1 절정**

**역사 사건:**
- 정조 7년, 사도세자를 장헌세자(莊獻世子)로 추존
- 수원 화성(華城) 축조 계획 발표 — 아버지의 영혼을 달래려는 의도

**세계관 사건:**
| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 1: 정조의 추존 — 해원의 착각 |
| **기간** | 1776~1783년 (7년) |
| **사도의 판단** | "조기 해제 조건에 근접했다" → 나가사에게 계약 종료 통보 |
| **나가사의 반박** | "기록된 것과 기억되는 것은 다르다. 네 죽음의 진실은 여전히 금기다" |
| **사도의 깨달음** | "기록이 아닌 살아있는 인간의 증언이 필요하다" |
| **원혼 수집** | 정조 시대에 숙청된 노론 강경파 희생자들, 장용영(정조의 친위대) 병사들의 원혼 수집 |
| **에너지 축적** | 0% → **5%** |

**핵심 갈등:**
- 사도: "아들이 나를 기억한다"
- 나가사: "기억된 것이 아니라 정치적 복권일 뿐이다"
- 결과: **조기 해제 첫 번째 실패**

### 【1800년】 정조 갑작스러운 사망 → **에피소드 2 시작**

**역사 사건:**
- 정조 갑작스러운 죽음 (의혹 많음)
- 11세의 순조(純祖) 즉위
- 정순왕후(貞純王后) 섭정 시작

**세계관 사건:**
- 사도의 원혼, 이중 충격: "아들을 잃었고, 조기 해제 가능성도 잃었다"

### 【1800~1820년】 세도정치 시대 → **에피소드 2 전개**

**역사 사건:**
- 안동 김씨 세도정치 본격화
- 삼정(전정·군정·환정) 문란 심화
- 억울한 죽음 폭발적 증가

**세계관 사건:**

| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 2: 세도정치의 어둠 — 나가사의 첫 방해 |
| **기간** | 1800~1820년 (20년) |
| **나가사의 방해** | 1) 사도가 수집하려는 원혼을 먼저 찾아가 저승으로 강제 송치 2) 저승사자에게 뇌물 바치며 협력 유도 |
| **사도의 반응** | 원혼을 소멸시킨 나가사를 목격 → 첫 직접 충돌 |
| **갈등 고백** | 사도: "이것이 계약 위반이다" / 나가사: "에너지 균형 문제다" |
| **원혼 수집** | 세도정치에 의해 유배·처형된 청류 관료들, 삼정 수탈로 아사한 농민들의 원혼 수집 |
| **에너지 축적** | 5% → **15%** (느린 증가, 나가사 방해로 인한 손실 포함) |

**핵심 발견:**
- 나가사가 사도의 에너지 축적 속도를 "의도적으로 조절"
- 계약이 나가사에게 일방적으로 유리함을 의심

---

### 【1811년】 홍경래의 난(洪景來亂) → **에피소드 3 절정**

**역사 사건:**
- 순조 11년, 평안도 출신들의 신분 차별에 항거
- "관서(關西)에서는 과거를 봐도 벼슬을 못 한다" → 민중 봉기
- 홍경래·우군칙 지도, 정주성(定州城) 거점 5개월 항전
- 진압 후 수천 명 처형 또는 전사

**세계관 사건:**

| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 3: 홍경래의 난 — 평안도 원혼 대량 흡수 |
| **기간** | 1811년 (단기 집중) |
| **사도의 반응** | 수천 명의 원혼이 한 번에 방출됨 → 평안도로 달려감 |
| **공명 경험** | 홍경래 원혼: "왕의 나라에 사는데 왕의 백성이 아니었다" / 사도: "왕의 아들이었는데 왕에게 버려졌다" |
| **원혼 흡수** | 처음으로 수백 개 원혼을 단시간에 흡수 → 감정적 과부하 위험 |
| **저승사자 개입** | "정주성 원혼들은 내 관할이다. 당장 돌려라" |
| **사도의 제안** | "이들의 기억을 내가 보존한다. 120년 후 세상에 전달하는 것이 이들의 진정한 해원이다" |
| **협상 결과** | 저승사자, 마지못해 30일 유예 승낙 |
| **에너지 축적** | 15% → **30%** (가장 큰 단계 점프) |

**핵심 성과:**
- 사도와 저승사자의 관계가 **협상의 구조**로 확립
- 원혼 기억 수집 방식의 체계화 (30일 한계)
- 매개자 개념의 씨앗 심어짐: "120년 후 세상에 전달"

### 【1839년】 기해박해(己亥迫害) → **에피소드 4 절정**

**역사 사건:**
- 헌종 5년, 조선에 천주교 신앙 확산
- 정하상·유진길 등 천주교 신도 수십 명 처형
- 프랑스인 선교사 3명(앵베르 주교, 모방·샤스탕 신부) 참수
- 천주교 신앙의 순교자 정신 형성

**세계관 사건:**

| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 4: 기해박해 — 신념을 위해 죽는 자들 |
| **기간** | 1839년 (단기 집중) |
| **특수성** | 순교자들은 억울하지 않음 — "사명의 미완성"으로 이승에 남음 |
| **사도의 혼동** | "이들은 억울하지 않은데 왜 이승에 있는가?" "한(恨) 에너지가 아닌 다른 에너지다" |
| **에너지 충돌** | 사도가 순교자 원혼을 흡수 시도 → 에너지 충돌 발생 → 사도의 원혼 불안정화 |
| **깨달음** | "모든 원혼 기억을 수집할 수 있는 것은 아니다" |
| **새로운 방식** | 흡수가 아닌 "교환"으로 관계 맺음 |
| **매개자 개념** | **처음으로 구체화**: 원혼의 기억을 이승의 인간에게 전달하는 역할 발명 |
| **에너지 축적** | 30% → **45%** |

**핵심 성장:**
- 원혼의 다양성 인식 (한(恨)만이 아닌 신념, 사랑의 에너지)
- 매개자 역할의 개념화: "미래의 인간이 원혼의 기억을 증언할 것"

### 【1862년】 임술 농민 봉기(壬戌農民蜂起) → **에피소드 5 절정**

**역사 사건:**
- 철종 13년, 삼정 문란에 저항하는 전국적 민란
- 경상도 진주에서 2월 시작 → 5월까지 71개 군현 확산
- 수백 명이 봉기 중 또는 진압 과정에서 사망

**세계관 사건:**

| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 5: 임술 농민 봉기 — 조기 해제 시도의 실패 |
| **기간** | 1862년 (단기 집중) |
| **사도의 성찰** | 100년 방랑 돌이켜보기 — "1762년 내 죽음과 1862년 농민의 죽음은 구조가 같다" |
| **역사적 깨달음** | "이들 모두 국가 폭력의 피해자다" (구조적 인식) |
| **조기 해제 2차 시도** | 살아남은 양반 기록자 이모(가명)를 선택 → 꿈을 통해 사도세자 이미지 전달 → 기록 유도 |
| **시도 실패** | 이모에게 수백 개 원혼 기억을 한 번에 전달 → 인간의 뇌가 감당 불가 → 이모 정신 붕괴 |
| **나가사의 조언** | "특별히 준비된 매개자가 필요하다. 그 준비에는 120년이 필요하다" |
| **사도의 수용** | 처음으로 나가사의 논리에 일부 동의 (그러나 현대의 가능성은 아직 모름) |
| **원혼 수집** | 임술 봉기 사망자 수백 명의 기억 집중 |
| **에너지 축적** | 45% → **70%** |

**핵심 변곡점:**
- 조기 해제 2차 실패로 120년 완성의 필연성 확인
- "특별히 준비된 인간 매개자" 필요성 깨달음 (하지만 현대에서 준비될 것임은 아직 모름)

### 【1866년】 병인박해(丙寅迫害)·병인양요(丙寅洋擾) → **에피소드 6 절정**

**역사 사건:**
- 고종 3년, 흥선대원군의 천주교 대탄압
- 프랑스 선교사 9명 참수, 조선 신도 8,000여 명 처형
- 프랑스의 병인양요(調선 침략 시도) 발생
- 강화도에서 조·불 전투 → 조선 승리, 프랑스군 격퇴
- **그러나 외규장각 도서 340권 약탈**

**세계관 사건:**

| 항목 | 내용 |
|------|------|
| **에피소드** | 에피소드 6: 병인박해·병인양요 — 시대의 단층선 |
| **기간** | 1866년 (단기 집중) |
| **원혼 규모** | 8,000명의 박해 사망자 원혼 — 방랑 최대 규모 |
| **역사 기록 소실** | 외규장각 도서 약탈: 임오화변 관련 기록 일부 프랑스로 반출 |
| **사도의 관찰** | "기록이 소실되는 것을 본다" |
| **최종 깨달음** | **"기록만으로는 진실이 보존되지 않는다. 살아있는 인간의 기억과 증언만이 진실을 영구히 보존할 수 있다"** |
| **저승사자의 경고** | "서양의 신(神) 개념이 들어오면서 조선의 원혼 법칙도 변하고 있다. 네 계약 기간이 끝나기 전에 모든 것이 바뀔 수 있다" |
| **원혼 수집** | 병인박해 희생자들의 기억 일부를 교환 방식으로 수집 |
| **에너지 축적** | 70% → **80%** |

**핵심 인식:**
- 시대 변화의 가속화 감지 (서양 영향, 원혼 법칙의 변동)
- 기록 vs 증언의 대립에서 **증언의 절대성** 확인
- 긴박감 증가: "시간이 부족하다"

### 【1876년】 강화도 조약(조일수호조규) → **에피소드 7 시작**

**역사 사건:**
- 고종 13년, 조선이 일본과 맺은 불평등 조약
- 개항의 시작 (부산, 인천, 원산)
- 서양 문물과 사상의 본격적 유입 시작

**세계관 사건:**
- 급격한 사회 변화 속에서 이승-저승의 경계 불안정화
- 서양 종교·사상의 유입으로 기존 무속 영계 구조 흔들림
- 조선의 원혼 법칙이 서서히 변형되기 시작

---

### 【1876~1882년】 개화기의 마지막 방랑 → **에피소드 7 절정**

**역사 사건:**
- 개항 이후 개화 정책 본격화
- 서양 문물의 대량 유입
- 민족주의와 개화주의의 갈등 시작
- 조선 사회의 급격한 변화

**세계관 사건:**

| 항목 | 내용 |
|------|------|

### 【1882년 임오군란】 차원문 개방

**역사 사건:**
- 고종 19년, 음력 7월 23일
- 군대의 급료 체불 + 민씨 여인천하 정치 비판 → 반란군 봉기
- 흥선대원군이 다시 정권 장악
- 일본군이 출병하여 진압 지원

**세계관 사건:**
- 1762년 임오화변 이후 정확히 120년(2갑자)
- 사도의 방랑 에너지가 100% 달성
- **차원문 개방 조건 충족**
- 미래의 매개자가 차원문을 통해 1882년 조선으로 진입할 준비 완료
